# roadmap-manage (lite — Cline / Llama 8B)

## Rules

- **One step per reply.**
- No `git commit` / `git push` / `git merge`.

## Preconditions

`docs/ROADMAP.md` or offer to create minimal sections: Done / In Progress / Pending / Backlog.

## Step 1

Read roadmap. Ask what to do (add, move, reprioritize). `[PAUSE]`

## Step 2

Apply change to `docs/ROADMAP.md`. Use checkboxes `[x]` / `[ ]`, dates, branch notes as in project style. Confirm in **3 bullets**. `[PAUSE]`
